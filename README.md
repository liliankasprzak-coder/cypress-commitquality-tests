## Instalação

```bash
git clone https://github.com/liliankasprzak-coder/cypress-commitquality-tests.git
cd cypress-commitquality-tests
npm install
```

## Execução

```bash
npx cypress run --config specPattern="cypress/e2e/**/*.cy.js"
```
